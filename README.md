# JavaFX Day-and-Night Scene

## Overview
This project is an interactive JavaFX application that simulates a simple residential scene transitioning between daytime and nighttime. The scene includes a house, roof, door, windows, ground, and background created with JavaFX shapes, colors, and coordinate-based positioning.

## My Contribution
I designed and programmed the full scene using Java and JavaFX. I created the visual layout with rectangles, circles, and polygons, then added a user-controlled slider with event listeners to adjust the background and window lighting based on user input.

## Technologies Used
- Java
- JavaFX
- GUI programming
- Event-driven programming

## Features
- Creates a residential scene using JavaFX shapes
- Uses custom colors and coordinate-based positioning
- Includes a slider for user interaction
- Transitions the scene between daytime and nighttime
- Changes window lighting based on the slider position

## How to Run
1. Make sure Java and JavaFX are installed.
2. Compile the program with JavaFX included.
3. Run the `DayAndNight` class.

Example command:

```bash
javac --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls DayAndNight.java
java --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls DayAndNight
